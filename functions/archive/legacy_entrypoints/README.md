# Legacy Entrypoints Archive

This folder contains historical Cloud Functions entrypoint variants that are no longer part of the live deployment path.

Do not deploy from these files.

Canonical deployment uses:

- `functions/package.json` -> `"main": "src/index.js"`
- `functions/src/index.js`

Archived here:

- `index.js`
- `stripe.js`
- `index_v2.js`
- `index.js.backup`
- `src_v2_backup/`
