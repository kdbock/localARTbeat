# Tools

This directory is the preferred home for one-off engineering utilities that do
not belong in the project root.

Put here:

- migration helpers
- data extraction scripts
- analysis utilities
- ad hoc maintenance scripts that are still useful

Do not put here:

- production app code
- canonical release runbooks
- long-lived product documentation

When moving existing root utilities into this folder, do it in small batches and
verify any documented workflow that references them.
