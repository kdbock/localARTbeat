# Local ARTbeat

Local ARTbeat is a Flutter app focused on a simple local-art loop:

- capture public art
- discover nearby art with radar/map views
- share activity into the community feed
- earn ranking progress through captures, discoveries, art walks, and engagement
- submit community events
- submit local business sponsorships for review

The mobile app intentionally no longer carries marketplace, commissions, messaging, artist storefront, artwork sales, old ads, or admin dashboards. Moderation/admin work belongs outside the mobile app.

## Active Packages

- `artbeat_auth`
- `artbeat_core`
- `artbeat_capture`
- `artbeat_art_walk`
- `artbeat_community`
- `artbeat_events`
- `artbeat_profile`
- `artbeat_settings`
- `artbeat_sponsorships`

## Common Commands

```sh
flutter pub get
flutter analyze lib
flutter run
```

The current iOS simulator target used during the cleave is `iPhone 17`.
